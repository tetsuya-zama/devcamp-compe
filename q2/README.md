# FizzBuzz問題

## 問題
0から100までの数字が一行ずつ入力されます。以下の通り変換して出力してください。

* 3の倍数の場合・・・'Fizz'
* 5の倍数の場合・・・'Buzz'
* 3の倍数でも5の倍数でもある場合・・・'FizzBuzz'
* それ以外の場合・・・数値をそのまま半角で出力

### 入力例
一行目の入力はデータの行数を表します。その後、与えられた行数だけ0~100までの数値が続きます。

``` 
5 #データは5行
0
30
1
2
6
```

### 出力例
データの行数だけ結果を出力してください。

```
0
FizzBuzz
1
2
Fizz
```
